# Plataforma para el Diseño de una Operación Táctica Interoperable

## DESCRIPCIÓN

Plataforma web y Android para diseñar, coordinar y visualizar operaciones tácticas en tiempo real con mapas, rutas, personal, vehículos y avisos centralizados.

## OBJETIVO

Facilitar la planeación y el monitoreo operativo mediante información georreferenciada compartida entre web y dispositivos móviles.

## METODOLOGÍA

1. Diagnóstico: necesidades operativas, rutas, recursos y zonas.
2. Diseño: arquitectura web, Android, API y base de datos.
3. Implementación: mapa, tracking, chat, avisos, rutas y objetos tácticos.
4. Optimización: pruebas de conexión, sincronización y rendimiento.
5. Evaluación: simulaciones, validación de datos y revisión de resultados.

## ¿QUÉ NOS DIFERENCIA?

Permite planear operaciones sobre mapas interactivos, integrando rutas, zonas, cuadrículas y recursos en una sola vista operativa.

Conecta la plataforma web con dispositivos móviles para que el personal en campo y el centro de mando trabajen con la misma información.

Centraliza registros, avisos y eventos históricos para dar seguimiento a cada operación y facilitar la revisión posterior.

## RESULTADOS

Se logró una plataforma funcional para registrar operaciones, asignar recursos, visualizar zonas, consultar rutas y sincronizar información entre la web y la app Android.
